ซอฟต์แวร์นี้เป็นผลงานที่พัฒนาขึ้นโดย นาย ธรรมพล อึ้งตระกูล จาก มหาวิทยาลัยราชภัฏอุบลราชธานี ภายใต้การดูแล
ของ อาจารย์ สัณชัย ยงกุลวณิช ภายใต้โครงการ อาจอง - มหาสงครามศึกแผ่นดินผู้กล้า ซึ่งสนับสนุนโดย ศูนย์เทคโนโลยีอิเล็กทรอนิกส์
และคอมพิวเตอร์แห่งชาติ และสำนักงานส่งเสริมอุตสาหกรรมซอฟต์แวร์แห่งชาติ โดยมีวัตถุประสงค์เพื่อส่งเสริมให้นักเรียนและนักศึกษาได้
เรียนรู้และฝึกทักษะในการพัฒนาซอฟต์แวร์ ลิขสิทธิ์ของซอฟต์แวร์นี้จึงเป็นของผู้พัฒนา ซึ่งผู้พัฒนาได้อนุญาตให้ศูนย์เทคโนโลยีอิเล็กทรอนิกส์และคอมพิวเตอร์แห่งชาติ
และ สำนักงานส่งเสริมอุตสาหกรรมซอฟต์แวร์แห่งชาติ เผยแพร่ซอฟต์แวร์นี้ตาม "ต้นฉบับ" โดยไม่มีการแก้ไขดัดแปลงใดๆ
ทั้งสิ้น ให้แก้บุคคลทั่วไปได้ใช้เพื่อประโยชน์ส่วนบุคคลหรือประโยชน์ทางการศึกษาที่ไม่มีวัตถุประสงค์ในเชิงพาณิชย์ โดยไม่คิดค่าตอบแทนการใช้ซอฟต์แวร์
ดังนั้นศูนย์เทคโนโลยีอิเล็กทรอนิกส์และคอมพิวเตอร์แห่งชาติ และสำนักงานส่งเสริมอุตสาหกรรมซอฟต์แวร์แห่งชาติ จึงไม่มีหน้าที่ในการดูแล บำรุงรักษา จัดการอบรมการใช้งาน
หรือพัฒนาประสิทธิภาพซอฟต์แวร์ รวมทั้งไม่รับรองความถูกต้องหรือประสิทธิภาพการทำงานของ ซอฟต์แวร์ ลอดจนไม่รับประกัน
ความเสียหายต่างๆ อันเกิดจากการใช้ซอฟต์แวร์ทั้งสิ้น
